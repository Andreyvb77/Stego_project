  # Stego_project
## "Система обнаружения стеганографически скрытой информации в цифровых и мультимедийных файлах с помощью нейронных сетей"

## Цель работы  

Целью выпускной квалификационной работы является разработка и сравнительный анализ эффективности архитектур сверточных нейронных сетей *(CNN)* для обнаружения следов стеганографического внедрения (методом замены младших значащих битов — *LSB*) на основе методов глубокого обучения в растровых изображениях, а также выбор оптимальной модели из современных архитектур нейронных сетей для практического применения в системах стеганоанализа.

## Решаемые задачи  

1.	 Провести обзор существующих методов стеганоанализа и архитектур нейронных сетей, специализированных на обнаружении скрытых данных *(SRNet, Yedroudj-Net, Zhu-Net, GSR-Net)*.
2.	 Подготовить набор данных для верификации работоспособности моделей: датасет «Кошки и собаки» (1000 изображений, конвертированных в черно-белый формат) для задачи классической классификации.
3.	 Сгенерировать синтетический набор данных (256 изображений размером 256х256 пикселей с градиентной заливкой) и внедрить в них скрытое сообщение методом *LSB-replacement* для тестирования чувствительности моделей.
4.	 Реализовать программный модуль стеганоанализа на основе выбранных нейросетевых архитектур с использованием фреймворков глубокого обучения (например, *PyTorch* или *TensorFlow*).
5.	 Провести серию экспериментов:
- контрольная классификация изображений (кошки/собаки) для подтверждения обучаемости моделей.
      -  детектирование LSB-вложений на синтетических градиентных изображениях.
- детектирование LSB-вложений на стандартном датасете (1000 изображений, цифровая стеганография).
6.	Выполнить сравнительный анализ метрик качества *(Accuracy, ROC-AUC, Precision, Recall, F1-score)* для каждой модели и определить наилучшую архитектуру для задачи обнаружения LSB-стеганографии. 

## Характеристика работы 

1.	Актуальность работы 
В условиях стремительного роста объемов мультимедийного трафика в сети Интернет цифровые изображения становятся одним из наиболее популярных контейнеров для организации скрытых каналов передачи информации. Традиционные методы стеганоанализа, основанные на статистических признаках, часто оказываются неэффективными против современных алгоритмов внедрения и больших объемов данных.
Актуальность темы обусловлена необходимостью перехода к методам обнаружения на основе глубокого обучения *(Deep Learning)*, которые способны автоматически выявлять слабые статистические аномалии, вносимые в файлы при скрытии информации. 

Особенно важным является исследование устойчивости современных архитектур (включая механизмы внимания) к простейшим, но массово используемым методам, таким как *LSB-replacement*, что позволяет оценить базовый потенциал нейросетевых детекторов.
В работе используется метод стеганографии:
- *LSB-replacement* - внедрение информации в младшие биты пикселей изображения.

Для решения задачи применяются следующие архитектуры нейронных сетей:
- *SRNet (Steganalysis ResNet)* - специализированная глубокая архитектура для стегоанализа;
- *Yedroudj-Net* - одна из базовых моделей для анализа стеганографических искажений;
- *Zhu-Net* - современная архитектура с высокой эффективностью *(SOTA)*;
- *GSR-Net* - улучшенная модель на основе ResNet с механизмами внимания *(SE-Block)*, позволяющая усиливать значимые признаки.

2.	Теоретическая и практическая значимость:
Теоретическая значимость работы заключается в анализе особенностей стеганографического метода LSB-replacement, исследовании принципов работы глубоких сверточных нейронных сетей в задачах обнаружения слабых сигналов, сравнении специализированных архитектур стегоанализа *(SRNet, Zhu-Net, Yedroudj-Net, GSR-Net)*, изучении влияния различных типов данных на качество обучения моделей.
Практическая значимость работы заключается в разработке прототипа системы обнаружения скрытой информации в изображениях, реализации и сравнении нескольких архитектур нейронных сетей, формировании экспериментальной базы для оценки эффективности стегоанализа, возможности применения разработанной системы в задачах: цифровой криминалистики, анализа мультимедийного контента, обнаружения скрытых каналов передачи информации.
